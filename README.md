**MSSV**: **B2111834**

**Họ và tên**: **Đỗ Thanh Dũ**

## Requirements

-   **php >= 8.2.12**
-   **composer >= 2.7.7**

Clone project and run test:

```
git clone https://github.com/DoThanhDuB2111834/CT271_Project.git <project dir>
cd <project dir>
php artisan migrate
php artisan db:seed

php artisan serve
npm run dev
```
